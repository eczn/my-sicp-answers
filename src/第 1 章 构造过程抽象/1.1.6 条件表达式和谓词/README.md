# 习题 1.5 

``` scheme 
(define (p) (p))
(define (test x y)
    (if (= x 0)
        0
        y))

(test 0 (p))
```

如果是应用序，那么执行 `(test 0 (p))` 的时候会报错；因为需要马上得出 `(p)` 的结果来应用到 `test` 里，而正则序会展开计算式：

``` scheme
(test 0 (p))
;; => 
(if (= 0 0) 0 (p))
;; => 
0
```

因此得到结果 0，用这样的方式就可以判断解释器用何种求值序了。
